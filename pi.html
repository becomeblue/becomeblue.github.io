Hello Raspberry Pi

The Raspberry Pi is a computer that can be used to write code, surf the web, manage your files, and play games. You can plug in a mouse, keyboard and a screen if you want to, or connect to a Raspberry Pi over a network.

The Raspberry Pi sounds like a standard computer, and it is, but it is also much more. The Pi, as it is called, is a tiny computer the size of a credit card, that runs an operating system. It has input/output pins to control electronics and can be embedded into projects to control lights, sensors, motors, and can become the brain of a robot.
