Operating Systems

Software

A computer has stuff that tells it what to do, and is what we see on screen, or don't see. Software is a set of instructions that tells a computer's processor to do certain things.

Operating Systems

An operating system (OS) is software that manages hardware, other programs, and users. An OS is the foundation for everything a common user wants to do with their computer, and is one of the most, if not the most, important piece of software.

Common operating systems are Windows, OSX, Linux and BSD.
