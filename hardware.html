Hardware

A computer has something that processes information, somewhere to store information, and a way to power everything. Hardware is the collection of physical parts in a computer system that allows this to happen.

Computer Systems

A desktop is a computer that stays in one place, plugs into the wall and has a wired connection. You plug in a screen, keyboard and mouse. Inside a desktop is a motherboard, hard drive, disc drive, and power supply. A motherboard is a circuit board where you will find the CPU, RAM, ROM and connectors to the ports. Common ports found on a desktop: USB, video, audio, ethernet and power.

A laptop is a movable computer, with a battery, that connects wirelessly. It will have a screen, keyboard and trackpad built into it. Inside a laptop is a motherboard (CPU, RAM, ROM, WiFi module), hard drive, and battery. An external power supply is used with a laptop. A laptop also comes with a camera, microphone, and speakers. Common ports found on a laptop: USB, video, audio, memory card slot and power.

A smartphone is a hand-held computer, with a battery, that connects to mobile networks. Inside a smartphone is a circuit board containing a CPU/GPU, RAM, flash memory, GSM module, WiFi module and sim card slot. A smartphone also comes with a camera, microphone, speaker, and battery. Common ports found on a smartphone: audio, memory card slot, and USB.

A Raspberry Pi is a computer on a single board, the size of a credit card. You can plug in a screen, keyboard and mouse. On the board is a System on a Chip (CPU, GPU, RAM), memory card slot, input/output pins, and ports for USB, ethernet, video, audio and power. An external power supply is used with a Raspberry Pi.

A BeagleBone Black is a computer on a single board. You can plug in a screen, keyboard and mouse, however a BeagleBone Black is made to be plugged directly into your desktop or laptop via USB. On the board is a CPU, RAM, flash memory, input/output headers, and ports for USB, ethernet, video/audio, and power.

An Arduino is a microcontroller board. A microcontroller is a computer on a single chip. On the board is a microcontroller (CPU, RAM, ROM, flash memory), crystal oscillator, input/output pins, and ports for USB and power. An Arduino is powered over USB or with an external power supply.

Computer Parts

The most important parts in a computer system are as follows:

A central processing unit (CPU) is hardware that reads computer programs from memory and carries out the instructions. The CPU controls everything in the system.

Random-access memory (RAM) is a type of fast working memory, where all of the information is lost when powered is removed.

Read-only memory (ROM) is permanent memory used to store computer programs that are not to be changed, like the BIOS.

Flash memory or hard drives are types of slow memory that are used to store information after power is removed.

A disc drive is used to read and write to optical media such as CDs and DVDs.

A power supply converts power from the source and feeds power around the system.

Other common parts and terms to know are as follows:

GPU stands for graphics processing unit.

USB stands for Universal Serial Bus.

An SD card is a type of flash memory.

Ethernet is wired connectivity.

WiFi is wireless connectivity.

GSM stands for Global System for Mobile Communications.

A sim card is an integrated circuit used to connect to a mobile network.

GPIO stands for General Purpose Input/Output.

HDMI is a connection for video and audio.

A crystal oscillator is an electronic circuit that uses a vibrating crystal to create a signal with a precise frequency.
