Robot Friend

A robot friend is somebody that follows instructions perfectly. Robot friends are cool because you get to program them, or you can be the robot. They are fun, and help you to learn about programming and how a computer follows instructions.

Grab a pencil, paper and a friend, and program a robot.

Print

Steps

  1. Will you stack cups, dance or set up an obstacle course?
  2. Decide together on the symbols or words you will use for your robot language.
  3. The programmer writes the code down on paper. The robot is not allowed to move without instructions.
  4. The robot reads the code and follows each instruction exactly, even if there is a mistake.
  5. If the code does not work, the programmer can fix the code and give it back to the robot.
  6. Swap the programmer and the robot, and create a new challenge.
